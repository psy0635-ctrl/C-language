## 📌 프로젝트 소개

**무인편의점 관리자**는 플레이어가 무인편의점의 관리자가 되어  
CCTV로 매장 상황을 확인하고, 다양한 사건에 대응하는 **선택형 시뮬레이션 게임**입니다.

최근 무인편의점, 무인카페 등 비대면 매장이 늘어나고 있다는 점에서 아이디어를 착안하였습니다.  
보안을 너무 강하게 하면 평판이 낮아지고, 느슨하게 대응하면 도난 위험도가 높아지는  
**균형 잡힌 판단**이 핵심입니다.

## 🎮 게임 방법

### 메인 메뉴

```
========================================
       [ 무인편의점 관리자 ]
========================================
  1. 게임 시작
  2. 게임 설명
  3. 최고 점수 확인
  4. 종료
========================================
```

### 진행 방식

1. 플레이어 이름 입력
2. 총 **5턴** 동안 랜덤 상황 발생
3. 각 상황마다 **1 / 2 / 3번** 중 하나를 선택
4. 선택 결과에 따라 4가지 상태값이 변화
5. 5턴을 버티면 최종 등급 획득

## 📊 상태값 시스템

| 상태값 | 초기값 | 설명 |
|--------|--------|------|
| 🔒 보안 점수 | 50 | 높을수록 좋음 |
| 💰 매출 | 100 | 높을수록 좋음 |
| ⭐ 평판 | 50 | 높을수록 좋음 |
| ⚠️ 도난 위험도 | 30 | **낮을수록 좋음** |

### 게임 오버 조건

- 도난 위험도 **≥ 100** → 실패
- 평판 **≤ 0** → 실패

### 최종 점수 계산식

```
최종 점수 = 보안 점수 + 매출 + 평판 - 도난 위험도
```

### 등급 기준

| 점수 | 등급 | 메시지 |
|------|------|--------|
| 200점 이상 | ★ S등급 | 매우 우수한 무인편의점 관리자입니다! |
| 150점 이상 | A등급 | 안정적으로 매장을 관리하였습니다. |
| 100점 이상 | B등급 | 기본적인 관리는 하였지만 보완이 필요합니다. |
| 100점 미만 | C등급 | 매장 관리에 어려움이 있었습니다. |

## 🔀 핵심 알고리즘: Fisher-Yates 셔플

8개의 이벤트를 중복 없이 랜덤으로 섞기 위해 Fisher-Yates 알고리즘을 사용하였습니다.

```c
void shuffleEvents(int arr[], int size) {
    int i, j, temp;

    // 배열을 0, 1, 2, ... 순서로 초기화
    for (i = 0; i < size; i++) {
        arr[i] = i;
    }

    // 뒤에서부터 랜덤 위치와 교환
    for (i = size - 1; i > 0; i--) {
        j = rand() % (i + 1);
        temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```
